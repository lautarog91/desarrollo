Lenguajes, herramientas que utilizaremos

Diseño ux-ui

Es la disciplina que se encarga de crear productos digitales que sean simples, intuitivos, eficientes, visualmente atractivos para el usuario.

Html

Se trata de un lenguaje de marcado de etiqueta, se encarga de establecer la estructura basica de la pagina web, aplicacion.

Css

Lenguaje web para aplicar formato visual al html,modificando color, tamaño, letras,margenes, etc.(Hojas de estilo en cascada).

javascript

Lenguaje de programacion para dar dinamismo, interactividad con el usuario a paginas web, aplicaciones.

React

Libreria, framework para crear interfaces de usuario, facilitando el desarrollo de aplicaciones de una sola pagina.
